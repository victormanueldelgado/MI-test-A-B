Página objeto del test: A u original

El código del test de Google Analytics ha decidido presentarte la versión A de la página cuya eficacia está siendo analizada. El objetivo de esta página es que el usuario haga visite el enlace que incluye más abajo. Si la página está bien diseñada debería:

  • facilitar que el usuario pueda encontrar el enlace y hacer click en él, no incluir obstáculos para que el usuario pueda realizar la acción deseada
  • además no sólo debe ponerlo fácil, sino hacerlo deseable, es decir, debe convencer y persuadir al usuario para que realice la acción

Objetivos:

Los objetivos de una página, aquello que se desea que los usuarios que la visitan realicen, pueden ser muy diversos. Algunos ejemplos:
 
  • prestar atención al contenido (durante un tiempo mínimo)
  • descargar un documento
  • visualizar un vídeo incluido en la página
  • añadir un producto a un carrito de la compra
  • registrarse (proporcionar el email) para recibir información
  • etc.
GA proporciona 2 mecanismos fáciles para definir objetivos (tiempo de la visita y visita a una página) y uno menos fácil que implica insertar código de seguimiento adicional en los elementos de la página con los que el usuario ha de interaccionar para cumplir el objetivo, para que dicho código envíe datos a GA cada vez que el usuario realiza con ese objeto la acción deseada.
